#  Reelz

An AI-powered movie recommendation app that suggests films based on your mood, preferences, or favourite genres — no account, no API key, just instant picks.

>  **Built with AI-assisted development** as a portfolio demo showcasing how quickly a polished, functional web app can be shipped using modern AI tooling.

---

## 🔗 Live Demo

**[thunrobi.github.io/Reelz](https://thunrobi.github.io/Reelz)**

---

## Features

- **AI-powered recommendations** — describe what you're in the mood for and get tailored movie suggestions
- **Static & fast** — no backend, no database, loads instantly
- **Clean UI** — minimal, focused interface designed for quick discovery
- **No sign-up required** — open and use immediately

---

## Tech Stack

- HTML, CSS, JavaScript
- Claude AI (Anthropic) for movie recommendations
- Hosted via GitHub Pages

---

## Running Locally

No build step needed:

```bash
git clone https://github.com/thunrobi/Reelz.git
cd Reelz
open index.html
```

## AI-Assisted Development

Reelz was designed and built in a single session using Claude (Anthropic) as a development accelerator — from concept to deployed app. This project is part of a series of portfolio demos exploring how AI tooling can be used to ship real, polished products rapidly.
